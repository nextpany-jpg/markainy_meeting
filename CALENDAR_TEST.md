# 📅 구글 캘린더 일정 등록 기능 테스트 가이드

## ✅ 구현 완료 사항

1. **백엔드**: Calendar API 통합 완료
   - `/api/create-calendar-event` 엔드포인트 추가
   - Calendar API 스코프 추가: `https://www.googleapis.com/auth/calendar`

2. **프론트엔드**: 자연어 일정 등록 기능
   - AI가 자연어에서 일정 정보 추출
   - 확인 UI 표시 후 캘린더에 등록
   - 성공 시 구글 캘린더 링크 제공

## 🔄 재인증 필요!

Calendar API 권한을 추가했으므로 **반드시 재인증**이 필요합니다:

1. 브라우저에서 `http://localhost:3000` 접속
2. "Gmail 로그인" 버튼 클릭
3. 구글 계정 선택
4. **새로운 권한 요청 화면에서 "캘린더 관리" 권한 승인**
5. 인증 완료

## 🧪 테스트 방법

### 1단계: 서버 시작
```cmd
cd C:\MCG_GIST_공공영업\해커톤\customer-research-system
node server.js
```

### 2단계: 브라우저에서 테스트
1. `http://localhost:3000` 접속
2. Gmail 재인증 (Calendar 권한 승인)
3. AI 자연어 검색창에 다음과 같이 입력:

**테스트 예시:**
```
3월9일 오전 11시 인사혁신처 구글캘린더 일정등록해줘
```

```
내일 오후 2시 삼성전자 미팅 등록
```

```
3월 10일 오전 10시 30분 LG화학 방문 일정 등록해줘
```

### 3단계: 확인
1. AI가 일정 정보를 추출하여 확인 화면 표시
2. "일정 등록" 버튼 클릭
3. 성공 메시지와 함께 구글 캘린더 링크 표시
4. 링크 클릭하여 구글 캘린더에서 일정 확인

## 🎯 지원되는 자연어 패턴

- **날짜**: "3월9일", "내일", "오늘", "2026-03-09"
- **시간**: "오전 11시", "오후 2시", "10시 30분"
- **장소**: "인사혁신처", "삼성전자 본사"
- **키워드**: "일정", "캘린더", "등록", "미팅", "회의"

## ⚠️ 주의사항

1. **Gemini API 키 필요**: AI 자연어 검색창 위에 API 키 입력
2. **재인증 필수**: Calendar 권한이 새로 추가되었으므로 반드시 재인증
3. **날짜 형식**: AI가 자동으로 파싱하지만, 명확한 날짜 표현 권장

## 🐛 문제 해결

### "인증이 필요합니다" 오류
→ Gmail 재인증 필요 (Calendar 권한 승인)

### "일정 정보를 추출할 수 없습니다" 오류
→ 더 명확한 날짜/시간 표현 사용
→ 예: "3월 9일 오전 11시 인사혁신처 미팅"

### API 키 오류
→ Gemini API 키 확인 (https://aistudio.google.com/app/apikey)

## 📝 작동 원리

1. 사용자가 자연어로 일정 요청 입력
2. AI가 "일정", "캘린더", "등록" 키워드 감지
3. Gemini AI가 일정 정보 추출 (제목, 날짜, 시간, 장소)
4. 확인 UI 표시
5. 사용자 확인 후 Google Calendar API 호출
6. 일정 생성 완료 및 링크 제공
