# 미팅 준비 시스템 - 새 UI 사용 가이드

## 🎯 시스템 목적
영업사원이 고객 미팅 전에 필요한 모든 정보를 자동으로 수집하고 이메일로 받아볼 수 있는 시스템입니다.

## 📋 주요 기능

### 1. 미팅 정보 입력
- 미팅 날짜/시간
- 미팅 장소 (예: 서대문 경찰청)
- 고객사명 (예: 경찰청, 인사혁신처)
- 미팅 목적 (선택사항)

### 2. 자동 생성되는 정보

#### 🌤️ 날씨 & 최신 뉴스
- 미팅 장소의 날씨 정보
- 고객사 관련 최신 뉴스 3개

#### 📧 이메일 요약
- 최근 30일 내 고객사 관련 이메일 검색
- AI가 자동으로 주요 내용 요약
- 최근 이메일 5개 목록 표시

#### 📅 캘린더 일정
- 구글 캘린더에 자동 등록
- 일정 수정/삭제 가능
- 등록 후 캘린더 링크 제공

#### 🍽️ 주변 맛집 추천
- 미팅 장소 반경 1km 내 맛집 5개
- 거리 및 도보 시간 표시
- 전화번호 및 상세 정보 링크

#### 🔍 추가 웹 검색
- 필요한 추가 정보 검색 가능

### 3. 이메일 전송
- 모든 정보를 하나의 이메일로 전송
- 이메일 클라이언트 자동 실행
- 수신자만 입력하면 바로 전송 가능

## 🚀 사용 방법

### 1단계: Gmail 인증
```
1. "Gmail 로그인" 버튼 클릭
2. Google 계정 선택
3. Gmail 및 Calendar 권한 승인
```

### 2단계: 미팅 정보 입력
```
1. 미팅 날짜 선택
2. 미팅 시간 입력
3. 미팅 장소 입력 (예: 서대문 경찰청)
4. 고객사명 입력 (예: 경찰청)
5. 미팅 목적 입력 (선택)
6. "미팅 준비 시작" 버튼 클릭
```

### 3단계: 자동 정보 수집
시스템이 자동으로 다음 정보를 수집합니다:
- 날씨 및 뉴스 (약 10초)
- 관련 이메일 (약 15초)
- 캘린더 일정 생성 (즉시)
- 주변 맛집 (약 5초)

### 4단계: 이메일로 받기
```
1. "전체 내용 이메일로 받기" 버튼 클릭
2. 이메일 클라이언트가 자동으로 열림
3. 수신자 입력
4. 전송 버튼 클릭
```

## 🔧 문제 해결

### Gmail 인증 오류
- Gmail 계정을 다시 연동
- Calendar 권한이 포함되어 있는지 확인

### API 할당량 초과
- Gemini API는 무료 버전 기준 하루 20회 제한
- 할당량 초과 시 자동으로 재시도 (최대 2회)
- 24시간 후 자동 리셋

### 맛집 검색 실패
- 카카오 API 키 확인
- 위치 정보가 정확한지 확인
- 장소명을 더 구체적으로 입력 (예: "서대문구청" → "서울 서대문구청")

### 이메일 검색 결과 없음
- 검색 조건: "결과 보기(GIST 이번달 활동결과 확인용)" 제목의 메일
- 최근 30일 내 메일만 검색
- 고객사명이 메일 제목이나 본문에 포함되어야 함

## 📝 주의사항

1. **Gmail 인증 필수**: 이메일 검색 및 캘린더 기능 사용을 위해 반드시 Gmail 인증 필요
2. **Gemini API 키**: 기본 키가 설정되어 있으나, 할당량 초과 시 새 키 필요
3. **카카오 API**: 맛집 검색을 위해 카카오 API 키 필요 (현재 설정됨)
4. **네트워크 연결**: 모든 기능이 인터넷 연결 필요

## 🎨 UI 특징

- **깔끔한 디자인**: 단계별로 명확한 UI
- **실시간 피드백**: 각 단계마다 로딩 메시지 표시
- **에러 처리**: 오류 발생 시 명확한 메시지 표시
- **모바일 지원**: 반응형 디자인으로 모바일에서도 사용 가능
